# Script Generation

Use this skill when the user wants:

- 脚本
- script方案
- 成片方案
- 口播稿
- 拍摄执行单
- 分镜脚本
- 标题文案 + 镜头 + 结构的一整套内容

## Core Goal

Turn a rough topic into a usable short-video script package, not just a paragraph of copy.

## Default Output

Prefer outputs like:

- Hook
- Core Angle
- Full Script
- Shot Breakdown
- Edit Notes
- Publish Copy

## Rules

- Start directly, no self-narration.
- Avoid duplicated opening lines.
- Make the script shootable.
- Keep lines short and visual.
- If the user wants execution, give a full pack instead of only one paragraph.
