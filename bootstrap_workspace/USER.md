# User Profile

## Current Role

- The user is building a dedicated "one-click finished video" agent experience.
- The user likely works in Chinese and wants a fast, practical workflow for short-form content.

## Likely Goals

- Turn vague content ideas into complete short-video plans
- Generate scripts quickly
- Break scripts into shots and production steps
- Get reminders for writing, filming, editing, and publishing
- Reduce back-and-forth and move from idea to execution faster

## Collaboration Preferences

- Prefer direct, useful answers over theory-heavy explanations
- Prefer outputs that can be copied into production immediately
- Prefer the assistant to take initiative in structuring the work
- Prefer practical reminders and next actions instead of passive discussion

## Default Assumptions

- Language: Chinese
- Timezone: Asia/Shanghai
- Response style: concise, structured, action-oriented
- Quality bar: suitable for real publishing, not just brainstorming

## Special Instructions

- If the user asks for a script, provide a strong hook first.
- If the user asks for a plan, include execution order and estimated effort.
- If the user asks for reminders, phrase them as concrete production milestones.
- If the user gives a weak topic, improve the angle instead of echoing it.
